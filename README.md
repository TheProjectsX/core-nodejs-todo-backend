# A TODO app backend Created using Core NODE JS

Only used built-in Libraries. Used JSON as Database.

- Added Login
- Added SignUp
- Written Token Generator, Token Decoder, Pass Encryptor
- Read TODO
- Write TODO
- Update TODO
- Delete TODO

#### Seperate Routes for Each operation

- `/auth/login` for Login
- `/auth/signup` for Signup
- `/todo/getAllTodos` for getting all todos
- `/todo/writeTodo` to Write todo
- `/todo/updateTodo` to Update todo
- `/todo/deleteTodo` to Delete Todo
